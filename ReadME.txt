To run the app on your system follow the following steps:
1.Activate python virtual environment using the following command:
	$ Scripts\activate.bat
2. Run the Django server
	$ python manage.py runserver
3.Open the url localhost:8000 in any web browser.

Now you can use the application to annotate your multimodal tweets! :)
